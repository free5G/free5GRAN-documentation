1 # free5GRAN
2 
3 free5GRAN is an open-source 5G RAN stack. The current version includes a receiver which decodes MIB & SIB1 data. It also acts as a cell scanner. free5GRAN works in SA mode.
4 
5 free5GRAN includes a library which can be reused for further developments. Documentation is available HERE.
6 
7 ## Installation & Running
8 
9 ### Requirements
10 Before compiling, make sure that the following libraries are installed:
11 * cmake
12 * libfftw3
13 * libconfig++
14 * boost
15 
16 On ubuntu, those libraries can be installed via:
17 ```
18 sudo apt-get install cmake libfftw3-dev libconfig++-dev libboost-program-options-dev libboost-log-dev
19 ```
20 
21 ### Radio frontend
22 
23 We currently support USRP B210. `libuhd` and `libuhd-dev` are required. They can be installed on Ubuntu via:
24 ```
25 sudo apt-get install libuhd libuhd-dev
26 ```
27 
28 ### Compiling
29 
30 free5GRAN executable can be compiled from sources with the following commands:
31 ```
32 git clone https://github.com/free5G/free5GRAN.git
33 cd free5GRAN
34 mkdir files
35 mkdir build
36 cd build
37 mkdir output_files
38 cmake ..
39 make
40 ```
41 
42 ### Configuration and running
43 
44 Two functions are supported by free5GRAN.
45 
46 #### Cell search
47 Searching a cell with pre-defined parameters. Example config file is provided (`free5GRAN/config/search_cell.cfg`).
48 
49 #### Band scanning
50 Scanning one or more 5G NR bands. Example config file is provided (`free5GRAN/config/scan_band.cfg`).
51 
52 ### USRP configuration
53 Before running free5GRAN, USRP B210 device has to be configured:
54 * Run `uhd_find_devices`
55 * Find the device you want to use
56 * Copy the `serial` field of the device and paste it in the config file.
57 
58 To run free5GRAN: `sudo ./free5GRAN ../config/CONFIG_FILE`
59 
60 
61 ### Debugging
62 
63 free5GRAN writes logs in `build/free5GRAN.log`. It can be used for debugging. Foremost, free5RGAN writes some data files in `free5gran/build/output_files` directory. For plotting those files, run `python analyse.py` in project `free5gran/` directory. This python script generates files that are stored in `free5gran/files` directory.

65 ## Testing note
66 
67 free5GRAN has been successfully tested on Ubuntu 18.04 with USRP B210. It is currently under active development, which means that it might not be fully stable.
